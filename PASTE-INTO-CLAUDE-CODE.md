# ATELIER Brand Manual Website — Project Summary

## Project Overview

## Features Implemented
- 9 sections (Hero, Esencia, Valores, Personalidad, Tono de Voz, Identidad Visual, Fotografia, Aplicaciones, Footer)
- Loading screen with pulsing logo
- Hero parallax scrolling effect
- Scroll animations (Intersection Observer)
- Hover-reveal value cards with centered text
- Click-to-copy color palette (HEX + CMYK)
- Sticky navigation with dot indicators
- Open Graph meta tags for social sharing
- Responsive design + prefers-reduced-motion support

## Typography
- **Playfair Display** — Headings
- **Montserrat** — Body text
